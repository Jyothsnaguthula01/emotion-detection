# Emotion Detection App

This is a simple emotion detection web app built using Streamlit and a trained ML model.

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the app:
streamlit run app.py

3. Open browser:
Go to [http://localhost:8501](http://localhost:8501) or the port shown in terminal.

## Requirements

- Python 3.10+
- Streamlit
- sklearn
- joblib
- nltk
